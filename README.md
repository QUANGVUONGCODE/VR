## ✨ Tính năng nổi bật

- 🌌 **Mô phỏng 3D:** Khám phá Trái Đất và cấu trúc Hệ Mặt Trời trong không gian 3 chiều sống động.
- 🥽 **Hỗ trợ Thực tế ảo (VR):** Trải nghiệm không gian ảo (WebXR) thông qua thiết bị VR trực tiếp trên trình duyệt.
- 🎨 **Giao diện hiện đại:** Xây dựng với React 19, Tailwind CSS v4 và các hiệu ứng chuyển động mượt mà bằng Framer Motion.

## 🛠 Công nghệ sử dụng

- **Frontend Core:** [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Vite 6](https://vitejs.dev/)
- **3D & VR:** [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/), [@react-three/drei](https://github.com/pmndrs/drei), [@react-three/xr](https://github.com/pmndrs/react-xr)
- **Styling & Animation:** [Tailwind CSS v4](https://tailwindcss.com/), [Motion (Framer)](https://motion.dev/), [Lucide React](https://lucide.dev/)

## 🚀 Cài đặt và Chạy ứng dụng

**Yêu cầu môi trường:**

- [Node.js](https://nodejs.org/) (Khuyến nghị phiên bản 20+ hoặc 22+)

### Các bước cài đặt

**1. Clone dự án và cài đặt thư viện**

```bash
npm install
```

**2. Cấu hình biến môi trường**

**3. Chạy môi trường phát triển (Development)**

```bash
npm run dev
```

Ứng dụng sẽ chạy tại địa chỉ: `http://localhost:3000` (hoặc IP Network hiện tại có hỗ trợ truy cập từ thiết bị VR).

## 📜 Các câu lệnh / Scripts

- `npm run dev` - Khởi động server phát triển tại port 3000.
- `npm run build` - Đóng gói dự án (production-ready) vào thư mục `dist`.
- `npm run preview` - Xem trước bản build production dưới local.
- `npm run lint` - Kiểm tra lỗi TypeScript mà không build.
- `npm run clean` - Xóa thư mục build cũ.
